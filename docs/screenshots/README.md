# Screenshots

Capture screenshots from the Jellyfin admin dashboard page:

`Dashboard -> Plugins -> Shirarium -> Shirarium Review`

Recommended files for release notes:

- `review-tab.png`
- `preflight-tab.png`
- `locks-tab.png`
- `history-tab.png`

Tips:

- Blur or crop any private filesystem paths before publishing.
- Use consistent window width so tables are readable.
- Prefer a seeded demo library for stable examples across releases.
